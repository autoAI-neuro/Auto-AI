\# Continuidad de Proyecto – AUTOAI





Este documento explica cómo \*\*retomar el proyecto en cualquier chat nuevo\*\* sin perder contexto.





\## Cómo retomar el contexto en un chat nuevo

1\. Abre un chat nuevo con el asistente.

2\. Copia y pega íntegro el contenido de `prompts/bootstrap\_context.txt`.

3\. A continuación, escribe: \*\*“Continuemos con el proyecto AUTOAI.”\*\*





> Sugerencia: guarda un acceso rápido a este archivo para no perder tiempo.
